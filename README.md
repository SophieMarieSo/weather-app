### 기능 정의

1. 앱이 실행되면, 현재 위치 기반의 날씨가 보인다
2. 날씨 정보에는 도시, 섭씨, 화씨, 날씨상태가 있다
3. 하단에는 5개의 도시 버튼이 있다 (현재 위치 + @)
4. 도시 버튼을 누르면, 도시별 날씨 정보가 나타난다
5. 현재 위치 버튼을 누르면, 다시 현재 위치 기반의 날씨가 나타난다
6. 데이터를 fetch되고 있는 동안, loading spinner가 나타난다
